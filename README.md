🔄 Palindrome Checker Application (Java)

📌 Overview

The Palindrome Checker Application is a Java project that demonstrates 13 different approaches to validating whether a string is a palindrome.

A palindrome is a word, phrase, number, or sequence of characters that reads the same forward and backward.

Examples
madam
racecar
level
radar

This project explores multiple Java programming concepts and data structures by implementing different algorithms to solve the same problem.

🎯 Objectives

Demonstrate multiple ways to solve a problem in Java

Practice data structures and algorithms

Improve understanding of recursion, collections, and string manipulation

Showcase clean modular Java programs
